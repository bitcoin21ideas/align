# Workflows

This directory contains n8n workflow definitions for the Align pipeline.

---

## Files

### `walks-processing.json`

The main pipeline workflow. Import this directly into n8n.

**What it does:**
1. Receives an audio file via webhook (triggered by iOS Shortcut)
2. Sends the audio to Deepgram Nova-3 for transcription
3. Reads `prompts/gemini-main.md` from this repo (live, at runtime)
4. Reads the current `handover/rolling-handover.md` from the storage repo
5. Assembles the Gemini prompt: today's date + rolling handover + transcript
6. Sends the prompt to Gemini Flash and receives four delimited outputs
7. Splits on `===ROLLING_UPDATE===`, `===ARCHIVE_ENTRIES===`, `===MEMORY_ENTRIES===`
8. Writes all four outputs to GitHub in parallel:
   - `walks/YYYY-MM-DD.md` — daily note (create or append)
   - `handover/rolling-handover.md` — overwrite
   - `handover/archive.md` — append
   - `memory/YYYY-MM.md` — append

**Two GitHub repositories are required:**
- **This repo** (Align) — read at runtime to fetch `prompts/gemini-main.md`
- **A private storage repo** — read/write for all walk outputs (walks/, handover/, memory/)

A single GitHub personal access token covering both repos is sufficient.

**Environment variables required:**

| Variable | Description |
|---|---|
| `DEEPGRAM_API_KEY` | Deepgram API key for Nova-3 transcription |
| `GEMINI_API_KEY` | Google AI API key for Gemini Flash |
| `GITHUB_TOKEN` | Personal access token with read access to this repo and read/write to your storage repo |
| `GITHUB_REPO` | Target storage repository (`owner/repo` format) |

**Gemini prompt:** managed in `prompts/gemini-main.md`. The workflow 
reads this file at runtime — do not edit the prompt inside the workflow 
JSON directly.

---

## Importing into n8n

1. Open your n8n instance
2. Go to **Workflows → Import from file**
3. Select `walks-processing.json`
4. Configure credentials for Deepgram, Google AI, and GitHub
5. Set the environment variables above
6. Activate the workflow

The webhook URL is generated by n8n on activation. Update your iOS 
Shortcut with this URL.

## Planned workflows

**`discord-capture.json`** — triggered by a message in `#quick-capture`. 
Runs the same pipeline with text substituted for the transcript, skipping 
the Deepgram step.

**`discord-query.json`** — triggered by a message in `#ask`. Reads 
memory and note files from GitHub, passes them to Gemini with a query 
prompt, and replies in Discord.

**`weekly-digest.json`** — scheduled Sunday run. Reads the past 7 daily 
notes and current month's memory file, posts a synthesis to `#weekly-digest`.

See [docs/discord-design.md](../docs/discord-design.md) for the design 
spec on all Discord workflows.
